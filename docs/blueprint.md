# **App Name**: Vikas Setu

## Core Features:

- Unified Government Schemes Directory: A comprehensive, browsable, and searchable catalog presenting both central and state government schemes with detailed descriptions and eligibility criteria.
- AI-Enhanced Scheme Search: An AI-powered search tool that intelligently processes natural language queries for scheme names, keywords, and state-specific contexts, providing relevant suggestions and advanced filtering capabilities.
- Public Opinion and Feedback System: A module enabling citizens to participate in 'Poll of the Day' topics, submit general feedback, and propose new ideas for government initiatives through an interactive wall.
- Scheme Rating and Engagement: Functionality allowing users to assign 5-star ratings to individual schemes, upvote/downvote their perceived effectiveness, and view schemes trending based on public interaction.
- Interactive State Development Map: A dynamic, color-coded map of India visualizing the progress of scheme implementation across states, providing detailed insights upon user interaction.
- National Progress Dashboard: A consolidated dashboard presenting overall and year-wise national development progress, incorporating key performance indicators derived from scheme implementation data.
- Data Management System: A robust database backend (PostgreSQL or MongoDB) to securely store and manage government scheme details, user feedback, ratings, and various progress metrics.

## Style Guidelines:

- Primary color (Saffron): #FF9933. Chosen for courage and national pride, it provides a vibrant highlight for interactive elements.
- Background color (White): #FFFFFF. Selected for peace and clarity, offering a clean canvas for content and visual information.
- Accent color (Green): #138808. Represents growth and prosperity, used for secondary interactive elements or positive indicators.
- Text color (Dark Navy): #1F2E4C. Evokes trust and authority, ensuring high readability against light backgrounds.
- Headlines font: 'Space Grotesk' (sans-serif), providing a modern, techy, and impactful feel suitable for 'India 2.0' messaging.
- Body font: 'Inter' (sans-serif), offering a neutral, objective, and highly legible experience for extensive informational content and descriptions.
- Utilize modern, clear vector icons, strategically incorporating relevant emojis for categorization and navigation to maintain a friendly, engaging, and clear visual language, consistent with the mockup.
- A clean and structured layout using card-based sections, hierarchical information display, and an implicit responsive grid system to adapt across devices, reflecting the provided wireframe's distinct areas.
- Incorporate subtle and functional animations, such as gentle hover effects on buttons and interactive elements, smooth transitions for search filters and data updates, and dynamic progress bar fillings to enhance user engagement.
